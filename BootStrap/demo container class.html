Q. What is container in boostrap? Container is like as box which has ability to store or manage the multiple HTML elements with padding. If we want to work with container using bootstrap or create container using bootstrap we have two types of container means two types of inbuilt classes to create container 1) .container : this class help us to provide fixed with responsive container means take some space at left hand side and space at right hand side like as margin

2) .container-fluid: this class provide full width container.
